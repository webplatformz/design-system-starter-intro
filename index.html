Project Reality

Setup

  • UX Designers working with Figma using the defined tokens
  • Design System with Lit & Storybook using the Figma tokens
  • Chromatic for automatic visual tests
  • React app with dependencies to the WebComponents and the Tailwind config

Processes

  • Design System gets published to the internal registry on tagging commit with a version
  • Chromatic is run on the main branch and changes get checked by UX
  • local linking of Web Components from the React app possible for development (npm link)

Experience

Good stuff:

  • setup easy to understand and use
  • consistent styling, low effort to build simple parts of the UI

Not great stuff:

  • missing changelog
  • hard to find/prevent side effects from changes sometimes in the app itself
